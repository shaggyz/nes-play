# Playing with NES technology

Some experiments about 6502 CPU and NES technology. Just for fun.

## Implemented components

- ROM Cartridge reader emulator. - WIP
- MOS 6502 CPU emulator. - WIP

## Compiling

```sh
make
```

## Running

```sh
./6502
```
